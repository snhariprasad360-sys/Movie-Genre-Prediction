# Movie Genre Prediction

## Overview
This project predicts the genre of a movie based on its description using Machine Learning and NLP techniques.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- TF-IDF Vectorization

## Files
- app.py
- best_model.pkl
- tfidf.pkl
- genres.json

## Dataset
The dataset is available in the Releases section of this repository.

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py
